# StockBuddy - AI Trading Assistant

StockBuddy is an AI-powered stock analysis assistant that helps users analyze stocks using real market data.

## Features
- Search stocks by company name
- Real-time stock analysis
- RSI, CAGR, and ROI calculations
- BUY / SELL / HOLD recommendations
- Chat-based UI

## Tech Stack
- Python
- Flask
- yfinance
- HTML / CSS / JavaScript

## Run Locally

pip install -r requirements.txt

python app.py
